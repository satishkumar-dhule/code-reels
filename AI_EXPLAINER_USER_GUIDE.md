## What is AI Explainer?

AI Explainer is a magical feature that uses advanced AI (Gemini or OpenAI) to explain any topic on the page in natural, conversational language. It can even read the explanation aloud in 9 different languages!

1. Click the **magical sparkle button** (âœ¨) at the bottom-right

## Troubleshooting

### "Please set your API key"
**Solution**: Open settings (âš™ï¸) and add your API key

### "Invalid API key"
**Solution**: 
- Check you copied the full key
- Verify key is active on provider's website
- For OpenAI: Ensure you have credits

### "No audio playing"
**Solution**:
- For Gemini: Check browser supports audio
- For OpenAI: Verify API key has TTS access
- Try regenerating

### "Explanation in wrong language"
**Solution**: 
- Open settings
- Select correct language
- Click "Save Settings"
- Regenerate explanation

### "Generation taking too long"
**Solution**:
- Check internet connection
- Try regenerating
- Switch to different AI provider
